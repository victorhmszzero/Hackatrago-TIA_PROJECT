# HackatAgro -TIA_PROJECT
O resultado do nosso projeto foi a produção de uma plataforma de gerenciamento agrícola TIA com base em um novo dispositivo criado para essa finalidade LUNA.
<img src="Assets/Banner.jpg" width = "100%">

## Conheça Luna
A **Luna** é um dispositivo que fará o controle de água, através do armazenamento da água e chuva para reutilização nas *lavouras*.    
Pensando no meio ambiente, foi desenvolvida com materiais sustentáveis. Um de seus aspectos é realizar a coleta de água e chuva, armazenando em um recipiente subterrâneo. Neste recipiente, a **Luna** irá fazer uma análise para ver a qualidade da água, indicando em qual estado ela está (boa ou ruim).

## Conheça TIA
<img src="Assets/DesktopPrint.jpg" width ="45%"> <img src="Assets/MobilePrint.jpg"  width ="45% ">.

A **TIA** é uma plataforma desenvolvida pela **Luna** com propósito de capacitar produtores e suas equipes de forma simples, prática e eficaz.<br>
Esses iriam desde irrigação até como a tecnologia pode ser aplicada em seu negócio, mostrando cases reais de grandes empresas.

### Objetivo do projeto
Este é um projeto elaborado para o **Desafio Expointer - HackatAgro 2022** (De 26 a 28 de agosto). O tema desse *HACKATHON* foi:
> "Apresentar soluções inovadoras e digitais para facilitar a disseminação de tecnologia e conhecimento para produtores e suas equipes”.

Essa maratona que desenvolver a criatividade, inovação e desenvolvimento de ideias colocando todos os conhecimentos em prática e aprendendo novas habilidades, em um período de 52h desde a divulgação do tema até ao horário de entrega.
